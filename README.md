# 萌萌校园圈 · 校园生活交友平台

> 《基于 Web 的编程》课程项目作品 — 题目一  
> 一个使用纯 **HTML5 + CSS3 + JavaScript（ES6, 原生）** 实现的校园社交互动平台。

## ✨ 项目特色
- **零依赖**：无任何前端框架（无 Vue / React / jQuery 等），打开 `index.html` 即可运行。
- **数据持久化**：使用 `LocalStorage` 模拟后端，刷新数据不丢失，便于本地演示。
- **完整的四大必做模块**：游客首页、用户系统、动态发布与浏览、互动与社交。
- **多个可选扩展**：私信、收藏、转发、夜间模式、可见范围、推荐算法、搜索、相册墙、热门榜单、跑马灯通知等。
- **响应式设计**：适配桌面、平板、手机多种屏幕。
- **管理员后台**：用户封禁/重置/删除/角色变更、动态/评论审核、数据看板。

## 🚀 快速开始

不需要任何构建步骤，只需：

```bash
# 1. 直接双击 index.html 用浏览器打开 即可
# 或者用 Python 起一个本地静态服务器（推荐，避免某些浏览器对 file:// 的安全限制）：
cd mtmtmty
python3 -m http.server 8080
# 然后浏览器访问  http://localhost:8080
```

## ✅ 功能清单

### ① 普通游客可访问首页
- 顶部 Logo / 导航 / 搜索 / 主题切换
- 跑马灯系统通知动效
- 最新动态时间流（无登录态自动隐藏点赞/评论入口）
- 仅展示「公开」可见的动态
- 右侧热门榜单（基于点赞数动态计算 → 满足"点赞较多自动更新"）
- 推荐同学栏 / 热门话题

### ② 用户系统
- 学号注册（6-12 位数字校验、重复检测、密码长度校验、二次密码确认、协议同意）
- 注册时可设置头像（预设头像 + 上传）、昵称、简介、兴趣标签
- 登录支持「记住我」「密码可见 / 隐藏」「忘记密码（重置）」
- 个人主页可查看 / 编辑昵称、头像、简介、兴趣标签、密码
- 个人主页展示：最近活跃时间、访客数、相册墙、关注 / 粉丝列表
- 管理员账号（admin.html）：封禁 / 解封、重置资料、设为/取消管理员、删除用户

### ③ 动态发布与浏览
- 发布页支持图文（500 字、最多 9 图）、本地上传 + 链接上传两种方式
- 表情面板（60+ emoji 一键插入）
- 自定义话题标签 + 内容自动识别 `#话题#`
- 可见范围：公开 / 仅好友（互关） / 仅自己
- 详情页：完整正文、点赞数、评论列表、时间戳、可见性、浏览量
- 删除（仅本人或管理员）、编辑（仅本人）/ 撤回（编辑后会标记"已编辑"）
- 评论支持文字 + 图片 + 表情回复

### ④ 互动与社交
- 浏览他人主页（公开/受限自动判断）
- 关注 / 取关、相互关注标识
- 主页展示粉丝/关注人数和"TA 的动态"
- 首页 Tab：全站 / 关注 / 热门
- 私信（UI 演示版，本地持久化）
- 转发（一键复制链接）、收藏
- 夜间模式（一键切换，CSS 变量驱动）

### ⑤ 个性化扩展
- 基于兴趣标签的动态推荐（发现页）
- 基于点赞 + 评论的热门榜单（首页/详情页/发现页）
- 全站搜索：动态 / 同学 / 话题 三 Tab
- 图片 Lightbox 预览
- 跑马灯滚动通知
- 响应式适配（桌面/平板/手机）
- 演示种子数据（首次访问自动生成 5 个用户、6 条动态、若干点赞/评论/关注）
